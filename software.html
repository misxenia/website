Software

Code

MATLAB package for the graph simulation and inference on sparse graphs with overlapping communities.

Graph simulation is based on an efficient way of simulating a random measure and the correspondning point process
Posterior inference is performed in Bayesian spirit using an MCMC sampler with Metropolis Hastings and Hamiltonian Monte Carlo steps.
The posterior sampler is linear per node, edge and feature. We use it to perfrom experiments on real datasets of social networks. For more details refer to the paper Exchangeable Random Measures for Sparse Graphs with Overlapping Communities , Adrien Todeschini, Xenia Miscouridou, Francois Caron arXiv:1602.02114 ,2017
